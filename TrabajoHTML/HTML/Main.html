Formato

Tipo de documento

Si queremos iniciar un documento html debemos de escribir la etiqueta: <!doctype html> con su correspondiente cierre: </html>


Formato de etiqueta

Etiqueta de apertura: <nombre de la etiqueta>

Etiqueta de cierre: </nombre de la etiqueta>
Si el elemento tiene atributos, estos no se incluirán en la etiqueta de cierre.


Etiqueta de cierre vs Elemento vacío

La etiqueta de cierre (</nombre de la etiqueta>) se usa para cerrar una etiqueta que va a contener texto dentro de ella.

Antes debemos de haber creado una etiqueta con ese nombre (<nombre de la etiqueta>), ya que si no sería errónea.


El elemento vacío (<nombre de la etiqueta/>) se usa para generar un elemento que solo va a tener atributos.


Atributos

Los atributos se representan como: <etiqueta atributo="valor del atributo">


Formato General


    <!doctype html> 
    <html lang="es">
    	 <head>
    	 </head>
    	 <body>
    	 </body>
    </html>

Título

La etiqueta de título se escribiría así:
<title>, cerrándola asi: </title> 


    <!doctype html> 
    <html lang="es">
        <head>
            <title>Título del documento HTML</title>
        </head>
        <body>
            Este es el texto del documento. 
        </body>
    </html>

Metadatos

Los metadatos se escriben asi:
<meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> 


    <!doctype html> 
    <html lang="es">
        <head>
            <meta charset=“utf-8”>
            <meta name=“author” content=“Iker”>
            <title>Título del documento HTML</title>
        </head>
        <body>
            Este es el texto del documento. 
        </body>
    </html>

<h3>Este es un título de tercer nivel</h3>
<h4>Este es un título de cuarto nivel</h4>
<h5>Este es un título de quinto nivel</h5>
<h6>Este es un título de sexto nivel</h6>

Secciones vs Artículos

Al usar <section>, vamos a crear un contenedor en el que se van a agrupar contenidos comunes

Al usar <article>, vamos a crear un contenedor en el que se van a agrupar contenidos algo más específicos

Al margen de esto, en la práctica su uso es el mismo, solo cambia su significado.

Documento

Encabezados

Los encabezados de una pagina son solamente de significado, por lo que aparecerá donde lo pongas.
<header>Encabezado</header>

Nav

Los nav, sirven para crear una lista de búsqueda.
Con nav y lista pasaria lo mismo que con section y article, que solo cambia su significado pero no su uso.
<nav>Menú</nav>

Aside

Los aside, sirven para indicar que esa parte está pensada para que vaya en un lado del documento.
<aside>Este texto iría en un lado</aside>

Footer

Los footer son el pie de página, y le pasa como al encabezado, que aparecerá donde se ponga.
<footer>Esto es un pie de página</footer>

Address

Los address sirven para marcar que en esa parte va a haber información del autor. <address>Aquí iria información del autor</address>

Agrupación

Párrafos

Los párrafos se usan para introducir texto genérico, es decir, sin especificar el tipo.
Se pueden introducir los párrafos que hagan falta en un mismo sitio. <p> Esto sería un párrafo</p>

Texto preformateado

Los textos preformateados sirven para que el texto introducido se visualice exactamente igual. <pre>Esto sería un texto preformateado <pre>

Listas

Las listas sirven para agrupar los contenidos de una parte.
Las listas <ul> no las numeran si no que ponen puntos a la izquierda (Ctrl + shift + 8)
Las listas <ol> numeran cada fila, poniendo el número correspondiente a la izquierda.
Los <dt> o dt sirven para definir un nombre en una lista.
Los <dd> o dd sirven para desarrollar una pequeña descripcion sobre el nombre del dt.

Elementos semánticos de texto

Resaltado

  • <i>Cursiva<i> = Itálica
  • <b>Negrita<b> = Negrita
  • <u>Subrayado<u> = Subrayado
  • <em>Enfasis<em> = Enfasis
  • <strong>Strong<strong> = Strong
  • <small>Small<small> = Small
  • <s>S<s> = palabra con s

Salto de Línea

Un salto de línea se escribe con <br>
y así se queda.

Hiperenlaces

Para usar un hiperenlace a una página externa hay que usar <a href="url">Nombre</a>
Para usar un hiperenlace a una parte del mismo documento HTML5 hay que usar <a href="#id">Nombre</a>
Para usar un hiperenlace a un archivo de la misma carpeta hay que usar <a href="direccionarchivo">Nombre</a>
Si se quiere abrir el link en una ventana nueva hay que usar <a href="https://es.wikipedia.org/wiki/Wikipedia:Portada" target="_blank">Wikipedia</a>

Simbolos especiales

Los simbolos especiales se escriben con un "&" seguido del nombre del símbolo.

Tablas

Las tablas se insertan asi:

  • <table> para abrir la tabla y </table> para cerrarla
  • <tr></tr> para insertar una fila en la tabla
  • <td></td> para insertar una columna en la fila de la tabla
  • <thead></thead> para aplicar un encabezado de tabla
